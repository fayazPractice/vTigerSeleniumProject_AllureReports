Manual Test Case: Create Contact With Organization

Test Objective: To verify that a user can create a new Organization, then create a Contact linked to that Organization, and validate both are created successfully.

Preconditions:

	•  User has valid login credentials.
	•  Application is accessible.
	•  Test data is available in the Excel file (sheet: Contacts).

Test Steps:

	1. Launch the application and log in with valid credentials.
	2. On the Home Page, click the "Organizations" link in the navigation menu.
	3. On the Organizations page, click the "Create Organization" button.
	4. In the Create Organization form: a. Enter a unique Account Name (from Excel row 1, column 3 + random number). b. Select an Industry from the dropdown (Excel row 2, column 3). c. Select an Account Type from the dropdown (Excel row 3, column 3).
	5. Click the "Save" button to create the Organization.
	6. Capture the Organization name displayed after creation.
	7. Verify that a success message appears and contains the expected text (Excel row 4, column 4).
	8. On the Home Page, click the "Contacts" link.
	9. On the Contacts page, click the "Create Contact" button.
	10. In the Create Contact form: a. Select a Salutation Type (Excel row 5, column 3). b. Enter First Name (Excel row 6, column 3). c. Enter Last Name (Excel row 7, column 3 + random number). d. Click the button to select an Organization.
	11. In the Organization selection popup: a. Switch to the Organization window (Excel row 8, column 3). b. Enter the previously created Organization name in the search box. c. Select the Organization filter type (Excel row 9, column 3). d. Click the "Search Now" button. e. Select the Organization from the search results. f. Switch back to the Contacts window (Excel row 10, column 3).
	12. Click the "Save" button to create the Contact.
	13. Verify that a success message appears and contains the expected text (Excel row 11, column 4).
	14. Log out of the application.
	15. Close the browser.

Expected Results:

	•  Organization and Contact are created successfully with correct details.
	•  All success messages match the expected values from the Excel data.
